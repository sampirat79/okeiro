# Okeiro - Senior TypeScript/Node Engineer - Medical Observation API Coding Test

## Server

### Requirements

- Node => 20.x or greater

### Installation

```sh
pnpm install
```

### Create env file

```sh
cp .env.sample .env
```

### Run

```sh
pnpm dev
open http://localhost:3000
```

### Run tests

```sh
pnpm test
```
